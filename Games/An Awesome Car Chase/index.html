This project was started to sharpen my Twine skills and after watching Transformers, I decided to create an action-paced game about the player stealing an alien device that gives them access to a futuristic car that allows them to fight off against goons who are after the device.

This Twine story was made in the Sugarcube story format and contains 20 passages and 31 links. It uses special Sugarcube display macros such as type to display the typewriter and uses custom Sugarcube code for the glitchy text and the blinky text.
